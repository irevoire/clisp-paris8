## What is this repo about

That's an ugly formatter for lisp. It follows the rules asked by the paris8 university.

## How to install

First, if it's not already done, you must install rust on your computer. See https://rustup.rs/
Then, run the following commands:

```
git clone https://github.com/irevoire/clisp-paris8
cd clisp-paris8
cargo install --path .
```

## How to use

You can run `clisp-paris8 the-file-you-want-to-format.lsp` to format a file.
